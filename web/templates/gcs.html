    Геозона

    Задайте межі поля на карті (обовʼязково перед стартом).

    Режим керування

    Автономний: маршрут по точках · Ручний: стрілки MAVLink

    Автономний маршрут

    Клік на карті — нова точка · перетягніть маркер · ✕ у списку — видалити

    Дані роботи (у JSON)

    Планувальник рядів (поле → GPS)

    План — чернетка. На сервер записуються реальні GPS дрона після завершення першого ряду; у ряду — RTK + CV hybrid.

    Контур поля: не задано. Додайте точки по краю поля (складна форма підтримується).

    Активне поле

    Експорт JSON — точки маршруту + дата/час роботи, оприскування, коментарі (файл на диск).

    Точка 1 = старт: після першого кліку маркер і дрон тут · 1→2→…→N

    Точки маршруту

      Ручне керування

      Утримуйте стрілки для руху · STOP — зупинка

      CV / оприскувач

      hybrid: ряд (depth) + YOLOv8 — людина/тварина/об'єкт попереду → стоп

      Журнал

      Запустіть: python main.py --full
