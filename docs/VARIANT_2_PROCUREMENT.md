# Варіант 2 — повний чекліст закупівлі заліза (з орієнтовними цінами)

**Схема:** RPi (CV) + **Pixhawk** (рух, GPS) + **станція** (ноутбук, GCS).  
**1 робот + 1 операторська станція.**  
Ціни **орієнтовні** (доставка, мито, курс, RAM-криза для RPi не враховані точно).  
Курс для орієнтиру в грн: **1 USD ≈ 41 UAH** (перерахунок округлений).

Офіційні магазини (перевіряйте актуальні ціни):
- [Holybro — Pixhawk, telemetry, GPS](https://holybro.com)
- [Luxonis — OAK-D](https://shop.luxonis.com)
- [Raspberry Pi — дистриб’ютори](https://www.raspberrypi.com/products/)

Покроковий монтаж і софт: [`VARIANT_2_SETUP.md`](VARIANT_2_SETUP.md).

## L. Що не купувати для варіанту 2

| Позиція | Чому |
|---------|------|
| Другий повний Pixhawk «про запас» на старті | Дорого; достатньо запасних кабелів |
| Jetson + RPi одночасно | Достатньо одного companion |
| 915 MHz radio в EU без перевірки | Можливі обмеження; 868 MHz |
| Дешевий GPS без компаса для FC | Поганий курс на rover |
| Камера лише веб без depth | Гірший hybrid planner |
